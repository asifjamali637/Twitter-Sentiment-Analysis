# Twitter Sentiment Analysis

## Overview
This project performs sentiment analysis on Twitter data using Machine Learning and Natural Language Processing (NLP).

## Dataset
Sentiment140 Dataset (1.6 Million Tweets)

## Technologies Used
- Python
- Pandas
- NLTK
- Scikit-Learn
- TF-IDF Vectorization
- Logistic Regression

## Workflow
1. Data Collection
2. Data Cleaning
3. Text Preprocessing
4. Stemming
5. TF-IDF Feature Extraction
6. Model Training
7. Sentiment Prediction

## Results
The model classifies tweets as Positive or Negative sentiments.
